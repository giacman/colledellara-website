# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-11-22

### Added
- Initial release of Colle dell'Ara coming soon website
- Responsive single-page design using Tailwind CSS
- Hero section with animated logo and mountain icon
- Three feature cards highlighting B&B amenities:
  - Immersed in Nature
  - Warm Hospitality
  - Authentic Breakfast
- Email subscription form for newsletter signup
- Contact section with email and phone information
- Social media links (Facebook, Instagram, WhatsApp)
- Smooth animations and hover effects
- Custom favicon with mountain/hill design
- English language content throughout the site
- Modern gradient backgrounds and styling
- Font Awesome icons integration
- Google Fonts (Playfair Display & Poppins)
- Vercel deployment configuration
- Comprehensive README documentation
- Git repository setup and GitHub integration

### Technical Details
- Built with vanilla HTML, CSS, and JavaScript
- Tailwind CSS via CDN (no build process required)
- Fully static site - no backend dependencies
- Mobile-first responsive design
- Cross-browser compatible
